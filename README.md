# design-skills

Web design toolkit for Claude Code — scroll animations, 3D effects, canvas, and component libraries.

## Skills

| Skill | What it does |
|-------|-------------|
| `animation` | Barba.js page transitions |
| `animation4` | Framer Motion — variants, gestures, layout animations |
| `scroll-animation-2` | Locomotive Scroll — smooth scroll, parallax |
| `scroll-animation-3` | GSAP + ScrollTrigger — timelines, scrubbing, pinning |
| `scroll-frame-sequence` | Canvas image-sequence scroll (Apple-style) |
| `2d-effekt` | PixiJS — WebGL 2D, particles, sprites |
| `3d-effekt-3` | Three.js + GSAP + R3F multi-library meta-skill |
| `3d-effekt-4` | Three.js — scenes, WebGL/WebGPU, materials |
| `3d-effekt-7` | React Three Fiber — declarative 3D in React |
| `3d-effekt-8` | Zdog, Vanta.js, Vanilla-Tilt — lightweight 3D |
| `webdesign-2` | UI/UX — 50+ styles, 161 palettes, 57 font pairings |
| `webdesign-3` | UI audit & Web Interface Guidelines review |
| `webdesign-4` | Magic UI + React Bits — pre-built animated components |
| `webdesign-5` | Modern web design trends 2024–2025 |
| `blender` | Blender → glTF export for Three.js / R3F |
| `21st-dev` | 21st.dev component library for landing page sections |

## Install

```bash
npx skills add github/arne-kill/design-skills-plugin -g -y
```

> Replace `arne-kill` with your actual GitHub username after pushing.
